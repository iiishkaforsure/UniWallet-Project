# UniWallet
**A Student Expense Tracking System**

## Overview
UniWallet helps students track daily expenses to avoid running out of pocket money. It saves data to a local file so records are never lost.

## Features
* **Add Expenses**: Log date, category, and cost.
* **View History**: See a summary of all spending.
* **Persistent Storage**: Uses file handling to save data.
* **Budget Alert**: Warns if spending exceeds Rs. 5000.

## Technologies Used
* Java (JDK 17)
* Object Oriented Programming
* File Handling (I/O)

## How to Run
1. Compile: `javac Main.java`
2. Run: `java Main`
